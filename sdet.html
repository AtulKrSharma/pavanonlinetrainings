SDET

Home SDET(Software Developement Engineer in Test)


Abstract


Nowadays, there has been a huge demand for SDET Engineers.Companies are expecting all round skills on various testing areas like GUI testing,API testing,Mobile testing and performance testing along with developement skills. This course is especially designed for testers who can work in multiple roles in software testing. You will learn about the Java programming, Selenium, Webserices/API Testing, Appium mobile app testing,Java scripting,Continuos Integration Process using Maven, Jenkins and Git, Github.


COURSE OUTLINE


  • Module 1: Overview on Automation

  • Module 2: Java Essentials for Selenium

  • Module 3: Java OOPS Concepts

  • Module 4: Selenium WebDriver

  • Module 5: TestNG

  • Module 6: Page Object Model & Page Factory

  • Module 7: Maven Integration with Selenium

  • Module 8: Automation Frameworks

  • Module 9: Continuous Integration (CI) - Maven, Jenkins & GIT

  • Module-10: Cucumber BDD Framework

  • Module-11: Appium for Mobile App Testing

  • Module-12: API/Webservices Testing

  • Module-13: JavaScript End to End Testing Framework Cypress

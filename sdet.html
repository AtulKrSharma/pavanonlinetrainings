Home SDET(Software Developement Engineer in Test)


Abstract


Nowadays, there has been a huge demand for SDET Engineers.Companies are expecting all round skills on various testing areas like GUI testing,API testing and Mobile testing along with developement skills. This course is especially designed for testers who can work in multiple roles in software testing. You will learn about the Java programming, Selenium, Webserices/API Testing, Appium mobile app testing,Continuos Integration Process using Maven, Jenkins and Git, Github.
